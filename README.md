# DevOps Health API

Simple Node.js API with CI/CD and deployment.

## Endpoint
GET /health

Response:
{
  "status": "ok"
}

## Run Locally
npm install
node app.js

## CI/CD
- Install dependencies
- Run test
- Validate build"# devops-health-api"
